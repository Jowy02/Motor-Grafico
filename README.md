## How to use
### Camera Contols

     * Mouse Wheel = Zoom in & Zoom out
     * Right Click = Free look & WASD movement
     * Alt + Left Click = Move around the object
     * F = Center camera on selected object
     * Left Shift = Multiply by 2 camera speed
